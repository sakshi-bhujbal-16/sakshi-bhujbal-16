## About Me

<img align="right" width="260" src="https://raw.githubusercontent.com/tandpfun/skill-icons/main/icons/AWS-Dark.svg"/>

I am an AWS DevOps Engineer based in Pune, India, currently building and operating cloud infrastructure at Greatway Software Pvt. Ltd. My work centers on designing reliable AWS environments, automating infrastructure with Terraform, and building CI/CD pipelines that move code from commit to production with minimal manual intervention. I spend most of my time close to the operational side of software — provisioning with EC2, VPC, S3, and IAM, containerizing applications with Docker and Kubernetes, and keeping systems observable through CloudWatch, Prometheus, and Grafana. Python and Bash scripting are how I close the gaps between tools, whether that means automating a deployment step or tightening a monitoring workflow. I hold a Bachelor's degree in Electronics and Telecommunication Engineering and have built my cloud and DevOps expertise through AWS-focused certifications and applied project work, with a continued focus on strengthening my grasp of distributed infrastructure and site reliability practices.
